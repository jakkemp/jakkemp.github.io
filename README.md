# jakkemp.github.io

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install --shamefully-hoist
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```

## Deployment

Generate the application for production:

```bash
npm run generate
```

Deploy dist to gh-pages:

```bash
npm run deploy
```
